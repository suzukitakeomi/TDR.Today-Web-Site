LINEでアトラクション待ち時間確認

TDR.Todayでは、LINEによるアトラクション待ち時間確認が可能なサービスを提供しております。この機能は、現在開発中で一部の方のみにご案内しています。ご利用希望の場合は、管理者までご連絡ください。

こんな機能欲しいとか、積極的に募集しております。
