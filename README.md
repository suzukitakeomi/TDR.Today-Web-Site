## 基本固定ページにデータ受信型サイトへ切替
TDR.Todayのサイト構造変更計画。サイトの軽量化及び転送量の削減。
* 現在、サーバー側のプログラムでHTMLデータを作成しているものを、基本の固定ページにjsonデータを読み込んで内容を変更していくタイプに変更。
* 一時的に、テスト導入もしていたので、方向性としては行けそう。
* jsonファイルの形式をサイトに合わせて生成するか検討(テストは元データそのままだった)。
* ajsxで自動更新とかもテストを行ったことあり(一部は現行で実施中)。
